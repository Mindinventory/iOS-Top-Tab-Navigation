## Requirements
- iOS 13.0+
- Xcode 13.0+

## Installation
To install it, simply add the following line to your Podfile:

```ruby
pod 'TopTabBarView', git: 'https://github.com/parthgohel2810/TopTabBarView-Framework.git', branch: 'main'
```
Then run `pod install` from the Example directory.
